# Uptrendis

Nuxt website for uptrendis.com

## Setup

Make sure to install dependencies:

```bash
# npm

# bun
bun install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash

# bun
bun run dev
```

## Production

Build the application for production:

```bash
# bun
bun run build
```

Locally preview production build:

```bash
# npm
npm run preview

# pnpm
pnpm preview

# yarn
yarn preview

# bun
bun run preview
```
